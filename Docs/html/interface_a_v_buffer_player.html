Detailed Description

Uses AVAudioPlayer to play a buffer of waveform data that you give it. You can use this for simple non-realtime sound synthesis.

Initializes the AVBufferPlayer.

Note:
Currently all output is 44100 Hz, 16 bit, mono.
Parameters:
buffer A buffer of floating point values between -1.0f and 1.0f that represent your waveform. The caller is responsible for freeing this buffer, which can be done immediately after this function returns.
frames The number of frames in the buffer. A frame is a single sample so the number of bytes in the buffer is 4 times the number of frames.
